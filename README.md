# Customer Support

Skills for the Customer Support department — Customer 360 lookups and brand-voice rules.

Install with [APM](https://microsoft.github.io/apm/):

```
apm install <owner>/mulesoft-customer-support-apm
```

Resolves every Skill and MCP server in `apm.yml` across any APM-aware harness (Claude Code, Cursor, Codex, Copilot, Windsurf).
